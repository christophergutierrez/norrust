<objective>
## Goal
Implement an analytic greedy AI in a new `norrust_core/src/ai.rs` module. The AI scores every
(move destination, attack target) pair using expected damage math — no RNG, no rollouts — and
applies the best option for each unit on its faction's turn. A headless AI-vs-AI integration test
verifies a full game terminates with a winner.

## Purpose
All game logic lives in Rust. The AI is another caller of `apply_action` — the same API used by
GDScript, tests, and future external agents. Keeping AI in `ai.rs` (not GDScript) means it can be
unit-tested, benchmarked, and swapped without touching the presentation layer.

## Output
- `norrust_core/src/ai.rs` — `ai_take_turn(state, faction)` + helper functions
- `norrust_core/src/lib.rs` — `pub mod ai` added
- `norrust_core/tests/simulation.rs` — `test_ai_vs_ai_terminates` integration test
</objective>

    use std::collections::HashSet;

    **`fn expected_outgoing_damage(attacker_attack_damage: u32, attacker_attack_strikes: u32,
        terrain_defense: u32, tod_mod: i32, resistance: i32) -> f32`**
    - hit_chance = (100.0 - terrain_defense as f32) / 100.0
    - effective_dmg = attacker_attack_damage as f32 * (100.0 + resistance as f32) / 100.0
    - per_strike = (effective_dmg * (100.0 + tod_mod as f32) / 100.0).max(0.0)
    - Returns: per_strike * hit_chance * attacker_attack_strikes as f32

    **`fn score_attack(attacker: &Unit, attacker_hex: Hex, defender: &Unit, defender_hex: Hex,
        state: &GameState) -> f32`**
    - tod = time_of_day(state.turn)
    - attacker_tod = tod_damage_modifier(attacker.alignment, tod)
    - defender_tod = tod_damage_modifier(defender.alignment, tod)
    - attacker_first_atk = attacker.attacks.first() — if None, return 0.0
    - defender_first_atk = defender.attacks.first() — if None, treat received as 0
    - defender_terrain = state.board.terrain_at(defender_hex).unwrap_or("")
    - attacker_terrain = state.board.terrain_at(attacker_hex).unwrap_or("")
    - defender_defense = defender.defense.get(defender_terrain).copied().unwrap_or(defender.default_defense) as u32
    - attacker_defense = attacker.defense.get(attacker_terrain).copied().unwrap_or(attacker.default_defense) as u32
    - atk_resistance = defender.resistances.get(&attacker_first_atk.attack_type).copied().unwrap_or(0)
    - def_resistance = attacker.resistances.get(&defender_first_atk.attack_type).copied().unwrap_or(0) (0 if no defender attack)
    - dealt = expected_outgoing_damage(attacker_first_atk.damage, attacker_first_atk.strikes, defender_defense, attacker_tod, atk_resistance)
    - received = if defender has attack { expected_outgoing_damage(def_atk.damage, def_atk.strikes, attacker_defense, defender_tod, def_resistance) } else { 0.0 }
    - kill_bonus = if dealt >= defender.hp as f32 { 3.0 } else { 1.0 }
    - Returns: dealt * kill_bonus / received.max(1.0)

    **`pub fn ai_take_turn(state: &mut GameState, faction: u8)`**
    Greedy move+attack for all units of `faction`, then EndTurn.
    Algorithm:
    1. Collect `unit_ids`: all unit IDs where `unit.faction == faction && !unit.attacked`
       (collect upfront to avoid borrow conflicts during apply_action)
    2. Collect `occupied`: all hex positions of units NOT belonging to `faction`
       plus friendly units other than the acting unit (recompute per unit during loop)
    3. For each uid in unit_ids (re-fetch from state each iteration — unit may be dead from retaliation):
       - Skip if unit no longer in state.units (killed by retaliation from a previous action)
       - Skip if unit.attacked (marked by retaliation)
       - start = state.positions[&uid]
       - zoc = get_zoc_hexes(state, faction)
       - candidates = reachable_hexes(&state.board, &unit.movement_costs, 1, start, unit.movement, &zoc, false)
       - all_occupied: set of positions of all units except uid
       - candidates: filter to hexes NOT in all_occupied (plus start is always valid — unit is already there)
       - best_score = 0.0; best_dest = start; best_target = None
       - For each candidate hex c in candidates:
         - For each (eid, enemy) in state.units where enemy.faction != faction:
           - epos = state.positions[&eid]
           - if c.neighbors().contains(&epos):
             - s = score_attack(unit, c, enemy, epos, state)
             - if s > best_score: best_score = s; best_dest = c; best_target = Some(eid)
       - If best_target is Some(eid):
         - If best_dest != start: apply_action(state, Action::Move { unit_id: uid, destination: best_dest })
           (ignore error — unit may have been blocked; still try to attack)
         - apply_action(state, Action::Attack { attacker_id: uid, defender_id: eid })
           (ignore error — target may have died from another attack this turn)
    4. apply_action(state, Action::EndTurn).expect("EndTurn must always succeed")

    Avoid:
    - Panicking on apply_action errors (attack target may have been killed by a prior unit this turn)
    - Holding borrows across apply_action calls (collect unit_ids and enemy data before mutating)
    - Processing units with attacked=true (already acted or hit by retaliation)

    In `norrust_core/src/lib.rs`, add `pub mod ai;` after the existing module declarations.
