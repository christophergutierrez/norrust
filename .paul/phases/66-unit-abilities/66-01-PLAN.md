<objective>
## Goal
Implement 3 unit abilities (leadership, regenerates, steadfast) in the Rust engine and add UI indicators for status effects (poisoned/slowed) and abilities in Love2D.

## Purpose
Completes the tactical mechanics layer — leadership makes faction composition matter (level hierarchy), regenerates gives trolls/woses survivability, steadfast rewards defensive positioning. UI indicators make status effects and ability bonuses visible to players.

## Output
- Unit struct gains `level: u8` field (needed for leadership level comparison)
- Leadership: +25% damage per level difference for adjacent lower-level allies
- Regenerates: self-heal N HP per turn (from "regenerates_N" ability string), cures poison
- Steadfast: double positive resistances when defending (being attacked)
- UnitSnapshot exposes level
- Visual indicators on unit circles for poisoned/slowed status
- Status effects shown in unit panel
</objective>

<acceptance_criteria>

## AC-1: Leadership grants +25% damage to adjacent lower-level allies
```gherkin
Given a unit with the "leadership" ability at level N
When an adjacent ally at level M < N attacks
Then the ally's effective damage is multiplied by (100 + 25 * (N - M)) / 100
And this applies to both initial attacks and retaliation
```

## AC-2: Regenerates heals per turn and cures poison
```gherkin
Given a unit with "regenerates_N" ability (e.g. "regenerates_8")
When EndTurn resolves for that unit's faction
Then the unit heals N HP (capped at max_hp)
And if the unit is poisoned, poison is cured (poison damage does NOT apply)
```

## AC-3: Steadfast doubles positive resistances when defending
```gherkin
Given a unit with the "steadfast" ability
When that unit is attacked (is the defender)
Then its positive resistance values are doubled (e.g. 20% → 40%)
And negative resistances (weaknesses) are not affected
```

## AC-4: Level field on Unit and UnitSnapshot
```gherkin
Given a unit spawned from a UnitDef with level=2
When get_state_json is called
Then UnitSnapshot includes level: 2
And level is copied at spawn and advance
```

## AC-5: Status effect indicators on unit circles
```gherkin
Given a unit with poisoned=true or slowed=true
When the board is drawn
Then a visual indicator appears on the unit circle (colored dot/symbol)
And the unit panel shows the status effect name
```

  <action>
    **Level field:**
    1. Add `level: u8` to Unit struct (default 1 in Unit::new())
    2. Wire level from UnitDef in place_unit_at (ffi.rs unit_from_registry) and advance_unit
    3. Add `level: u8` to UnitSnapshot, wire in from_game_state()

    **Leadership (+25% damage per level diff):**
    4. In Action::Attack handler, before computing effective_damage for attacker:
       - Check all adjacent hexes of attacker_pos for allies (same faction) with "leadership" ability at level > attacker.level
       - If found, compute highest level diff, multiply effective_damage by (100 + 25 * diff) / 100
       - Same for defender's retaliation: check adjacent allies of defender with leadership
    5. Add helper fn `leadership_bonus(state, unit_id) -> u32` returning bonus percentage (0, 25, 50, etc.)

    **Regenerates (self-heal per turn, cures poison):**
    6. In Action::EndTurn, for newly active faction's units (during healing phase):
       - Parse "regenerates_N" from abilities (extract N)
       - If found: heal N HP (capped at max_hp), AND set poisoned = false
       - Regeneration stacks with village healing

    **Steadfast (double positive resistances on defense):**
    7. In Action::Attack handler, when computing resistance modifier for the DEFENDER:
       - If defender has "steadfast" ability and resistance > 0, double it (cap at 100)
       - Also apply steadfast when the original attacker is being retaliated against (if attacker has steadfast)
    8. In simulate_combat: add steadfast modifier for defender resistance
       - Add leadership_bonus parameters (attacker_leadership_pct, defender_leadership_pct) to simulate_combat
       - Apply leadership damage bonus in simulation

    **Tests (add inline):**
    - test_leadership_boosts_adjacent_lower_level: level 2 leadership ally → +25% damage
    - test_regenerates_heals_per_turn: unit with "regenerates_8" heals 8 on EndTurn
    - test_regenerates_cures_poison: regenerating unit has poison cleared
    - test_steadfast_doubles_positive_resistance: 20% resistance → 40% when defending
    - test_steadfast_no_effect_on_weakness: -10% resistance stays -10%
    - test_level_on_snapshot: level field appears in UnitSnapshot
  </action>
