## Goal
Complete the tactical planning state machine with "attack from" comparison and terrain defense visibility. When re-ghosting to a different hex adjacent to the same enemy, auto-show the updated combat preview so the player can compare attack positions. Add attacker/defender terrain defense percentages to the combat preview panel.

## Purpose
The Phase 37-38 ghost + preview system lets players preview ONE attack. Phase 39 closes the loop by letting players compare attacks from different positions — the core tactical decision of hex-based combat: "which hex should I attack from?"

- Modified `CombatPreview` struct + FFI JSON: adds `attacker_terrain_defense` and `defender_terrain_defense` fields
- Modified `main.lua`: auto-preview on re-ghost when previous target still adjacent, terrain defense shown in preview panel
