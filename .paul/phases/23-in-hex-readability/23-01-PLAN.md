<objective>
## Goal
Render a unit's type name (abbreviated to first underscore-segment, max 7 chars, capitalized)
centered inside its hex circle at all times, so units are identifiable without clicking.

## Purpose
Players currently have to click every unit to learn its type. With 10+ units on board this
is friction. A glanceable label makes the game readable at a glance — core readability goal
of v1.0.

## Output
- Modified `_draw_units()` in `game.gd` draws a name abbreviation above HP text
- `def_id` abbreviation drawn centered at `center.y - 6` (font size 9)
- HP text shifted to `center.y + 7`, XP to `center.y + 19` to give vertical breathing room
- No Rust or snapshot changes (def_id already in UnitSnapshot JSON since Phase 6)
</objective>
