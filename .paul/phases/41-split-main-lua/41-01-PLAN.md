<objective>
## Goal
Split main.lua (~1,728 lines) into focused modules: hex.lua (pure math), draw.lua (all rendering), campaign_client.lua (scenario/campaign loading). main.lua retains state management, input handling, and Love2D callback dispatching.

## Purpose
Reduce main.lua from 1,728 to ~770 lines. Future edits (documentation, features) target specific modules instead of one monolith. Input handling stays in main.lua because it's the most tightly coupled to state mutation.

## Output
- norrust_love/hex.lua — pure hex math functions (no state dependency)
- norrust_love/draw.lua — all rendering functions
- norrust_love/campaign_client.lua — scenario and campaign loading helpers
- norrust_love/main.lua — slimmed to state + input + Love2D callbacks
</objective>
