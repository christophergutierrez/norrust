<objective>
## Goal

Extend `Board` with a per-hex terrain map, then implement A* pathfinding that
respects terrain movement costs and Zone of Control (ZOC). Units must stop when
they enter a ZOC hex unless they have the Skirmisher trait.

## Purpose

Pathfinding is required by `apply_action(Move)` to validate that a move follows
actual movement rules (terrain budget, ZOC). The current dispatcher accepts any
in-bounds destination; after this plan, it can call `find_path` to confirm the
move is legally reachable.

<acceptance_criteria>

## AC-1: Terrain Cost Accumulation

```gherkin
Given a Board where 3 consecutive hexes are all "forest" terrain
And movement_costs = {"forest": 1} and default_cost = 2
When find_path is called from start to the third hex with budget = 10
Then Some(path) is returned
And the returned cost equals 3
```

## AC-2: ZOC Blocks Path Beyond Enemy

```gherkin
Given a Board with a friendly unit at (0,0) and an enemy unit at (3,0)
And ZOC hexes include (2,0) (adjacent to the enemy)
When find_path is called from (0,0) to (4,0) with budget = 10 and is_skirmisher = false
Then None is returned  (unit cannot continue past the ZOC hex)
When find_path is called from (0,0) to (2,0) with same budget and is_skirmisher = false
Then Some(path) is returned (unit can reach the ZOC hex itself)
```

## AC-3: Skirmisher Bypasses ZOC

```gherkin
Given the same board setup as AC-2 (ZOC covers (2,0))
When find_path is called from (0,0) to (4,0) with budget = 10 and is_skirmisher = true
Then Some(path) is returned (skirmisher ignores ZOC stop rule)
```

      pub fn get_zoc_hexes(state: &GameState, moving_faction: u8) -> HashSet<Hex> {
          let mut zoc = HashSet::new();
          for (&unit_id, unit) in &state.units {
              if unit.faction != moving_faction {
                  if let Some(&pos) = state.positions.get(&unit_id) {
                      for n in pos.neighbors() {
                          zoc.insert(n);
                      }
                  }
              }
          }
          zoc
      }

    ── find_path ────────────────────────────────────────────────

    A* pathfinding with terrain movement costs and ZOC.

    Signature:

      pub fn find_path(
          board: &Board,
          movement_costs: &HashMap<String, u32>,
          default_movement_cost: u32,
          start: Hex,
          destination: Hex,
          movement_budget: u32,
          zoc_hexes: &HashSet<Hex>,
          is_skirmisher: bool,
      ) -> Option<(Vec<Hex>, u32)>

    Returns Some((path, total_cost)) where:
    - path is the full route from `start` to `destination` inclusive
    - total_cost is the total movement points spent
    Returns None if destination is unreachable within movement_budget respecting ZOC.

    Implementation:

    Data structures:
      // Min-heap: (f_cost, g_cost, hex). Reverse makes BinaryHeap a min-heap.
      let mut open: BinaryHeap<Reverse<(u32, u32, Hex)>> = BinaryHeap::new();
      let mut g_cost: HashMap<Hex, u32> = HashMap::new();
      let mut came_from: HashMap<Hex, Hex> = HashMap::new();
      let mut closed: HashSet<Hex> = HashSet::new();

    Heuristic (admissible):
      fn h(a: Hex, b: Hex) -> u32 { a.distance(b) }

    Seed:
      g_cost.insert(start, 0);
      open.push(Reverse((h(start, destination), 0, start)));

    Main loop:
      while let Some(Reverse((_, g, current))) = open.pop() {
          if current == destination {
              // Reconstruct path from came_from
              return Some((reconstruct_path(&came_from, start, destination), g));
          }
          if closed.contains(&current) { continue; }
          closed.insert(current);

          // ZOC stop rule: if we're at a ZOC hex and not a skirmisher,
          // this unit cannot continue — do not expand neighbours
          if zoc_hexes.contains(&current) && !is_skirmisher && current != start {
              continue;
          }

          for neighbor in current.neighbors() {
              if !board.contains(neighbor) { continue; }
              if closed.contains(&neighbor) { continue; }

              // Look up terrain cost for this neighbour
              let terrain_id = board.terrain_at(neighbor).unwrap_or("");
              let step_cost = movement_costs
                  .get(terrain_id)
                  .copied()
                  .unwrap_or(default_movement_cost);

              // 99 = impassable convention (matches UnitDef.movement_costs)
              if step_cost >= 99 { continue; }

              let tentative_g = g + step_cost;
              if tentative_g > movement_budget { continue; }

              if tentative_g < g_cost.get(&neighbor).copied().unwrap_or(u32::MAX) {
                  g_cost.insert(neighbor, tentative_g);
                  came_from.insert(neighbor, current);
                  let f = tentative_g + h(neighbor, destination);
                  open.push(Reverse((f, tentative_g, neighbor)));
              }
          }
      }
      None

    Path reconstruction helper (private):
      fn reconstruct_path(came_from: &HashMap<Hex, Hex>, start: Hex, end: Hex) -> Vec<Hex> {
          let mut path = Vec::new();
          let mut current = end;
          while current != start {
              path.push(current);
              current = came_from[&current];
          }
          path.push(start);
          path.reverse();
          path
      }

    test_terrain_cost_accumulation():
      // Straight-line path: (0,0) → (1,0) → (2,0) → (3,0), all "forest" (cost 1 each)
      let mut board = Board::new(10, 10);
      for col in 0..4 {
          board.set_terrain(Hex::from_offset(col, 0), "forest");
      }
      let mut movement_costs = HashMap::new();
      movement_costs.insert("forest".to_string(), 1u32);
      let zoc: HashSet<Hex> = HashSet::new();
      let result = find_path(
          &board, &movement_costs, 2,
          Hex::from_offset(0, 0), Hex::from_offset(3, 0),
          10, &zoc, false,
      );
      let (path, cost) = result.expect("path should exist");
      assert_eq!(cost, 3, "3 forest hexes × 1 mp each = 3");
      assert_eq!(path.first(), Some(&Hex::from_offset(0, 0)));
      assert_eq!(path.last(), Some(&Hex::from_offset(3, 0)));

    test_zoc_blocks_path_beyond():
      let board = Board::new(10, 10);
      let movement_costs: HashMap<String, u32> = HashMap::new(); // default_cost = 1
      // ZOC covers (2,0) — adjacent to enemy at (3,0)
      let mut zoc: HashSet<Hex> = HashSet::new();
      zoc.insert(Hex::from_offset(2, 0));
      // Cannot reach (4,0) — path would need to pass through ZOC hex (2,0)
      let blocked = find_path(
          &board, &movement_costs, 1,
          Hex::from_offset(0, 0), Hex::from_offset(4, 0),
          10, &zoc, false,
      );
      assert!(blocked.is_none(), "path past ZOC hex must be blocked");
      // Can reach the ZOC hex itself
      let to_zoc = find_path(
          &board, &movement_costs, 1,
          Hex::from_offset(0, 0), Hex::from_offset(2, 0),
          10, &zoc, false,
      );
      assert!(to_zoc.is_some(), "unit can enter ZOC hex");

      );
