## Goal

Implement combat resolution: seeded deterministic RNG, Time of Day (ToD) damage
modifiers, per-strike hit/miss resolution using terrain defense, and full
`apply_action(Attack)` wiring including dead-unit removal from GameState.

## Purpose

`apply_action(Attack)` is currently a no-op stub. This plan makes combat real:
attacks roll against terrain defense, ToD alignment modifiers apply, dead units
are purged. Every later system (integration test, AI hooks) depends on correct
combat math.

<acceptance_criteria>

## AC-1: ToD Damage Modifier Applied Exactly

```gherkin
Given an attacker with Chaotic alignment
And a 4-damage, 1-strike attack
And terrain defense = 0% (always hits)
When the attack is resolved during turn 5 (Night time of day)
Then the damage dealt is exactly 5  (4 × 125 / 100 = 5, i.e. +25%)
```

## AC-2: Hit Rate Matches Terrain Defense

```gherkin
Given a 60% terrain defense (40% hit chance)
And a seeded Rng (seed=1)
When 10,000 single-strike attacks are simulated
Then the hit count is between 3500 and 4500  (40% ± 5%)
```

## AC-3: Defender Death Purges Unit from GameState

```gherkin
Given an attacker (id=1, faction=0) with a 100-damage, 1-strike attack
And a defender (id=2, faction=1) with 30 HP at an undefended hex (0% defense)
When apply_action(Attack { attacker_id: 1, defender_id: 2 }) is called
Then the result is Ok(())
And state.units does NOT contain key 2
And state.positions does NOT contain key 2
```

    Maps a turn number to time of day using a 6-phase repeating cycle:
      // Phases: 0=Dawn(Neutral) 1=Midmorning(Day) 2=Afternoon(Day)
      //         3=Dusk(Neutral) 4=FirstWatch(Night) 5=SecondWatch(Night)
      pub fn time_of_day(turn: u32) -> TimeOfDay {
          match (turn.saturating_sub(1)) % 6 {
              1 | 2 => TimeOfDay::Day,
              4 | 5 => TimeOfDay::Night,
              _ => TimeOfDay::Neutral,
          }
      }

      pub fn tod_damage_modifier(alignment: Alignment, tod: TimeOfDay) -> i32 {
          match (alignment, tod) {

              (Alignment::Lawful, TimeOfDay::Night) => -25,

    Define Rng (Xorshift64 — deterministic, no external crate needed):

      pub struct Rng {
          state: u64,
      }

      impl Rng {
          /// Seed must be non-zero; panics otherwise.
          pub fn new(seed: u64) -> Self {
              assert!(seed != 0, "Rng seed must be non-zero");
              Self { state: seed }
          }

          pub fn next_u64(&mut self) -> u64 {
              let mut x = self.state;
              x ^= x << 13;
              x ^= x >> 7;
              x ^= x << 17;
              self.state = x;
              x
          }

          /// Returns true with probability hit_pct / 100.
          /// hit_pct >= 100 always returns true; hit_pct == 0 always returns false.
          pub fn roll_hit(&mut self, hit_pct: u32) -> bool {
              if hit_pct >= 100 { return true; }
              if hit_pct == 0 { return false; }
              (self.next_u64() % 100) as u32 < hit_pct
          }
      }

    Implement resolve_attack:

      /// Simulate `strikes` attack strikes.
      /// Each strike hits with probability (100 - terrain_defense_pct)%.
      /// On hit, damage is base_damage modified by tod_modifier (+/-25 or 0).
      /// Returns total damage dealt.
      pub fn resolve_attack(
          rng: &mut Rng,
          base_damage: u32,
          strikes: u32,
          terrain_defense_pct: u32,
          tod_modifier: i32,
      ) -> u32 {
          let hit_pct = 100u32.saturating_sub(terrain_defense_pct);
          // Apply ToD modifier to damage (integer arithmetic, min 0)
          let modified_damage = ((base_damage as i64 * (100 + tod_modifier as i64)) / 100)
              .max(0) as u32;
          let mut total = 0u32;
          for _ in 0..strikes {
              if rng.roll_hit(hit_pct) {
                  total += modified_damage;
              }
          }
          total
      }
