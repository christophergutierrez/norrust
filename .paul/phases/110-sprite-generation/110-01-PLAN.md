```gherkin
Given tree-structured units like "spearman/swordsman" in the UNITS dict
When generate_sprites.py --unit spearman/swordsman is run

And the unit_name key uses "/" separators matching the directory tree
```

```gherkin

```

    **1b. Add all missing unit definitions to UNITS dict**

    For each unit missing sprites (~97 units), add a tuple entry:
    `(description, melee_weapon, ranged_weapon_or_None, defend_desc)`

    Research each unit from Wesnoth lore to write accurate descriptions. Group by faction for readability:

    **Loyalists** (~20 missing): spearman evolutions (swordsman, royal_guard, pikeman, halberdier, javelineer), bowman evolutions (longbowman, master_bowman), cavalryman evolutions (dragoon, cavalier), heavy_infantryman evolutions (shock_trooper, iron_mauler), horseman + evolutions (lancer, knight, grand_knight, paladin), fencer + evolutions (duelist, master_at_arms), merman_fighter + evolutions, sergeant (already has sprites but verify)

    **Rebels** (~17 missing): elvish evolutions (marksman, sharpshooter, ranger, avenger, marshal, hero, champion, rider, outrider, druid, shyde, sorceress, enchantress, sylph), merman_hunter + evolutions, wose + evolutions, mage evolutions (red_mage, arch_mage, great_mage, silver_mage, white_mage, mage_of_light)

    **Northerners** (~20 missing): orcish evolutions (crossbowman, slurbow, slayer, nightblade, warlord), goblin_spearman + evolutions (impaler, rouser), troll_whelp + evolutions (troll, troll_warrior, troll_rocklobber), wolf_rider + evolutions (goblin_knight, direwolf_rider, goblin_pillager), naga_fighter + evolutions (naga_warrior, naga_myrmidon)

    **Undead** (~20 missing): skeleton + evolutions (revenant, draug, deathblade), skeleton_archer + evolutions (bone_shooter, banebow), ghost + evolutions (shadow, nightgaunt, wraith, spectre), ghoul + evolutions (necrophage, ghast), vampire_bat + evolutions (blood_bat, dread_bat), walking_corpse, soulless, dark_adept evolutions (dark_sorcerer, lich, necromancer)

    For evolutions, descriptions should show visual progression (better armor, larger weapons, more ornate gear, higher-level magic effects).
