- `norrust_love/main.lua` calls camera_mod functions instead of inline camera code

<task type="auto">
  <name>Task 1: Create camera_mod.lua</name>
  <files>norrust_love/camera_mod.lua</files>
  <action>
    Create camera_mod.lua module:

    1. Module-local refs set by init(): camera, hex, scn, get_viewport, clamp
    2. init(deps) — stores references from deps table
    3. apply_offset() — clamps camera.offset_x/y to min/max (was apply_camera_offset)
    4. center(reset) — centers camera on board, computes pan limits (was center_camera)
       - Uses hex.to_pixel, scn.COLS, scn.ROWS, get_viewport(), camera table
       - Calls apply_offset() at end
    5. update(dt) — handles arrow key panning + camera lerp (was inline in love.update lines 852-882)
       - Arrow key check: love.keyboard.isDown for left/right/up/down
       - Pan speed from camera.PAN_SPEED, lerp from camera.LERP_SPEED
       - Returns true if panning occurred (so love.update can early-return)

    Name the file camera_mod.lua (not camera.lua) to avoid collision with the camera state table variable name.
  </action>
  <verify>luajit -bl norrust_love/camera_mod.lua /dev/null — syntax check passes</verify>
  <done>AC-1 satisfied: camera_mod.lua provides all camera operations</done>
</task>

<task type="auto">
  <name>Task 2: Wire main.lua to use camera_mod</name>
  <files>norrust_love/main.lua</files>
  <action>
    1. Add `local camera_mod = require("camera_mod")` to requires
    2. Remove apply_camera_offset() function (lines 119-122)
    3. Remove center_camera(reset) function (lines 127-162)
    4. Add local wrappers that delegate:
       ```
       local function apply_camera_offset() camera_mod.apply_offset() end
       local function center_camera(reset) camera_mod.center(reset) end
       ```
       These wrappers preserve the local function names used throughout main.lua
       (in select_unit, build_campaign_ctx, love.resize, input.init ctx).
    5. In love.load, after state is initialized, call:
       camera_mod.init({camera = camera, hex = hex, scn = scn, get_viewport = get_viewport, clamp = clamp})
    6. In love.update, replace the arrow key panning block (lines 852-866) and
       camera lerp block (lines 868-882) with:
       ```
       camera_mod.update(dt)
       ```
       Remove the early return from panning (camera_mod.update handles both panning and lerp).
  </action>
  <verify>luajit -bl norrust_love/main.lua /dev/null — syntax check passes</verify>
  <done>AC-2 satisfied: main.lua delegates camera operations</done>
