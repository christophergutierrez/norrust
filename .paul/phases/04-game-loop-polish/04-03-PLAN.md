<objective>
## Goal
Apply unit resistance modifiers in combat (reducing or amplifying damage by attack type), and
display an informational HUD showing turn number, time of day, and active faction name.

## Purpose
Resistance modifiers have been deferred since Phase 2 — they complete the combat simulation
so damage output now matches the Wesnoth formula. The HUD eliminates the need to watch the
console to track whose turn it is or what time of day is active.

## Output
- `unit.rs`: `resistances: HashMap<String, i32>` field on Unit (default empty)
- `game_state.rs`: effective damage = `base_damage × (100 + resistance) / 100` applied before
  `resolve_attack()` in both the forward strike and retaliation
- `gdext_node.rs`: resistances copied from UnitDef in `place_unit_at()`; new
  `get_time_of_day_name()` bridge method
- `game.gd`: HUD drawn at top of screen: "Turn N · Day / Night / Neutral · Blue's Turn"
</objective>

<acceptance_criteria>

## AC-1: Resistance Modifiers Applied in Combat
```gherkin
Given a defender unit with resistances.blade = -25 (25% weakness to blade)
When an attacker with a blade attack of damage 8 strikes
Then each hit deals floor(8 × 125 / 100) = 10 damage (not 8)
  And retaliation damage is similarly modified by the attacker's resistances
```

## AC-2: HUD Shows Turn, Time of Day, Active Faction
```gherkin
Given the game is running
When _draw() renders the board
Then a HUD is visible at the top of the screen showing:
  "Turn N · [Day|Night|Neutral] · Blue's Turn" or "Red's Turn"
  And pressing E (end turn) updates the HUD immediately on the next draw
```

## AC-3: HUD Updates Correctly Across Turns
```gherkin
Given turn 1 (Day), blue active → "Turn 1 · Day · Blue's Turn"
When blue presses E (end turn)
Then the HUD shows "Turn 1 · Day · Red's Turn" (still turn 1 until both factions end)
When red presses E
Then HUD shows "Turn 2 · Neutral · Blue's Turn" (turn increments, phase changes)
```

       ```rust

           let defender = state.units.get(&defender_id).unwrap();
           defender.resistances.get(&attack.attack_type).copied().unwrap_or(0)
       };
       let effective_damage =
           ((attack.damage as i64 * (100 + resistance as i64)) / 100).max(0) as u32;
       ```
       Then pass `effective_damage` instead of `attack.damage` to `resolve_attack()`:
       ```rust
       let damage = resolve_attack(&mut state.rng, effective_damage, attack.strikes, terrain_defense, tod_mod);
       ```

       resistance to the defender's attack type. After cloning `def_attack`, add:

       ```rust
       let ret_resistance = {
           let a = state.units.get(&attacker_id).unwrap();
           a.resistances.get(&def_attack.attack_type).copied().unwrap_or(0)
       };
       let ret_effective_damage =
           ((def_attack.damage as i64 * (100 + ret_resistance as i64)) / 100).max(0) as u32;
       ```
       Pass `ret_effective_damage` instead of `def_attack.damage` to `resolve_attack()`.

  <action>
    **In `norrust_core/src/gdext_node.rs`:**

    1. Add `TimeOfDay` to the existing combat import:
       ```rust
       use crate::combat::{resolve_attack, time_of_day, tod_damage_modifier, Rng, TimeOfDay};
       ```

    2. Add a new `#[func]` after `get_turn()`:
       ```rust
       /// Returns "Day", "Night", or "Neutral" for the current turn's time of day.
       /// Returns "Neutral" if no game exists.
       #[func]
       fn get_time_of_day_name(&self) -> GString {
           let turn = self.game.as_ref().map(|s| s.turn).unwrap_or(1);
           match time_of_day(turn) {
               TimeOfDay::Day     => "Day",
               TimeOfDay::Night   => "Night",
               TimeOfDay::Neutral => "Neutral",
           }.into()
       }
       ```

    **In `norrust_client/scripts/game.gd`:**

    3. At the end of `_draw()`, after the win overlay block, add the HUD:
       ```gdscript
       # 6. HUD: Turn · Time of Day · Active Faction
       if not _game_over:
           var faction_name = "Blue" if _core.get_active_faction() == 0 else "Red"
           var tod = _core.get_time_of_day_name()
           var hud_text = "Turn %d  ·  %s  ·  %s's Turn" % [_core.get_turn(), tod, faction_name]
           draw_string(
               ThemeDB.fallback_font,
               Vector2(10, 20),
               hud_text,
               HORIZONTAL_ALIGNMENT_LEFT,
               -1, 14, Color.WHITE
           )
       ```

    Avoid: modifying `_input()`, `_setup_rust_core()`, or any other draw block.
    Do not hide the HUD on game over — the win overlay already blocks it; skip HUD then.
