<task type="auto">
  <name>Task 1: Add hex stencil masking to draw_terrain_hex</name>
  <files>norrust_love/assets.lua</files>
  <action>
    Update `assets.draw_terrain_hex()` to use Love2D stencil for hex clipping:

    1. Define a stencil function that draws a filled hex polygon
    2. Before drawing the terrain image:
       - love.graphics.stencil(stencilFn, "replace", 1)
       - love.graphics.setStencilTest("greater", 0)
    3. Draw the scaled terrain image (existing code)
    4. After drawing:
       - love.graphics.setStencilTest()  -- clear stencil test

    The stencil ensures the rectangular PNG is clipped to the hex boundary.
    The fallback polygon path does NOT need stencil (it's already hex-shaped).

    Implementation note: The stencil function captures cx, cy, hex_radius,
    and hex_polygon_fn via closure. Use "replace" action and value 1.
  </action>
  <verify>Game launches without stencil errors (test with existing colored polygons — no PNGs yet)</verify>
  <done>AC-1 satisfied: hex stencil masking implemented</done>
</task>
