<objective>
## Goal
Convert flat `scenarios/*.toml` files into per-scenario directories (`scenarios/<name>/board.toml`, `units.toml`, `dialogue.toml`). Update all references in Rust loaders, Lua code, campaign TOML, and tests.

## Purpose
A contributor creating a new scenario should find all related files in one directory — board, units, triggers, and dialogue. This mirrors the Phase 70 unit content merge pattern.

## Output
- 5 scenario directories: contested/, crossing/, ambush/, night_orcs/, final_battle/
- Each contains board.toml, units.toml, and optionally dialogue.toml
- Campaign TOML paths updated
- Dialogue path derivation changed: same directory as board, named dialogue.toml
- All Rust tests updated with new paths
- SCENARIOS table in main.lua updated
</objective>
