
<task type="auto">
  <name>Task 1: Create sound module and generate placeholder sounds</name>
  <files>norrust_love/sound.lua</files>
  <action>
    Create `sound.lua` module that:

    1. Generates simple placeholder sound effects using love.audio.newSource with SoundData:
       - `hit`: short noise burst (~0.1s)
       - `miss`: softer/higher swoosh (~0.1s)
       - `death`: lower thud (~0.2s)
       - `move`: light tap (~0.05s)
       - `recruit`: ascending tone (~0.15s)
       - `turn_end`: bell/chime tone (~0.2s)
       - `select`: subtle click (~0.03s)

    2. Provide API:
       - `sound.load()` — generate all SoundData and create Sources
       - `sound.play(name)` — play a named effect (clone source for overlap)
       - `sound.set_volume(v)` — set master volume 0.0-1.0
       - `sound.get_volume()` — return current volume
       - `sound.toggle_mute()` — toggle mute on/off
       - `sound.is_muted()` — return mute state
       - `sound.play_music(path)` — load and loop a music file (or nil to stop)
       - `sound.stop_music()` — stop current music

    3. Use love.audio.newSource("static") for effects, "stream" for music
    4. Store volume/mute state in module-level locals
    5. Generate sounds procedurally with SoundData (sine waves, noise) so no external files needed

    Avoid: depending on external .ogg/.wav files for effects (generate procedurally).
    Avoid: complex DSP — simple waveforms are fine for placeholders.
  </action>
  <verify>luajit -bl norrust_love/sound.lua /dev/null — syntax check passes</verify>
  <done>AC-1, AC-2, AC-3, AC-4 infrastructure ready</done>
</task>

<task type="auto">
  <name>Task 2: Wire sound triggers into main.lua</name>
  <files>norrust_love/main.lua</files>
  <action>
    1. Add `local sound = require("sound")` at top of main.lua
    2. Call `sound.load()` in love.load()
    3. Wire sound effects via events bus and direct calls:
       - In execute_attack callback: play "hit" if damage > 0, "miss" if damage == 0, "death" if unit killed
       - In move_anim on_complete or move initiation: play "move"
       - After successful recruit: play "recruit"
       - On end turn (where events.emit("turn_end") is): play "turn_end"
       - On unit selection: play "select"
    4. Add keybindings in love.keypressed:
       - M key: sound.toggle_mute()
       - Minus key: sound.set_volume(sound.get_volume() - 0.1)
       - Equals/Plus key: sound.set_volume(sound.get_volume() + 0.1)
    5. On scenario load: check if scenario has a music field, call sound.play_music() or sound.stop_music()

    Avoid: playing sounds during AI turn animations (keep AI fast and quiet, or only play combat sounds).
    Avoid: breaking existing control flow — sound calls are fire-and-forget additions.
  </action>
  <verify>luajit -bl norrust_love/main.lua /dev/null — syntax check passes</verify>
  <done>AC-1, AC-2, AC-3, AC-4, AC-5 satisfied</done>
</task>
