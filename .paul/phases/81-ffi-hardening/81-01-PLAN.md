<objective>
## Goal
Replace 7 unsafe `unwrap()` calls in ffi.rs with proper error returns, and unify manual JSON construction with consistent quote escaping.

## Purpose
The FFI layer is the boundary between Rust and Lua. Panics from `unwrap()` crash the Love2D process with no recovery. Replacing them with graceful error returns makes the game robust against edge cases. Consistent JSON escaping prevents malformed output from data containing quotes or backslashes.

## Output
- `norrust_core/src/ffi.rs` — all unwrap() replaced with early returns, JSON construction unified
</objective>
