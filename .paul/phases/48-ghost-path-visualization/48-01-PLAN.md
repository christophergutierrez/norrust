<objective>
## Goal
Show the hex-by-hex path from unit's current position to the ghost position during ghost movement, so the player can see the route their unit will take.

## Purpose
Currently ghost movement teleports the preview — the player sees where the unit will end up but not how it gets there. Visualizing the path improves tactical awareness (terrain traversal, ZOC awareness).

## Output
- New FFI function `norrust_find_path` exposing Rust's A* pathfinder
- Lua wrapper in norrust.lua
- Ghost path computed and stored when ghost position changes
- Path drawn as connected hex highlights between unit and ghost
</objective>
