    **2. Setup prompt zoom correction (lines 236-245):**
    The prompt position uses `bx + ctx.board_origin_x + ctx.camera_offset_x` but doesn't account for zoom. The board-space coordinates (bx, by) and camera_offset need to be scaled by zoom:
    ```lua
    local zoom = ctx.camera_zoom or 1
    local sx = ctx.board_origin_x + zoom * (bx + ctx.camera_offset_x)
    local sy = ctx.board_origin_y + zoom * (by + ctx.camera_offset_y)
    ```
    This matches the camera transform: translate(origin) → scale(zoom) → translate(offset).
