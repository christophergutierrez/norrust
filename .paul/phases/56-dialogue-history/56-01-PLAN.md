Add a dialogue history log that accumulates all triggered dialogue during a scenario. Accessible via key press, scrollable, dismissible.

## Output
- `dialogue_history` array accumulating all fired dialogue entries
- 'H' key toggles history panel overlay
- Scrollable history panel in draw.lua
- History clears on scenario load (fresh per playthrough)
