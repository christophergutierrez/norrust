<tasks>

<task type="auto">
  <name>Task 1: Enable fullscreen in conf.lua</name>
  <files>norrust_love/conf.lua</files>
  <action>
    Set `t.window.fullscreen = true` and `t.window.fullscreentype = "desktop"` in love.conf().
    Desktop fullscreen uses the monitor's native resolution while keeping the window borderless —
    this is preferred over exclusive fullscreen which can cause resolution switching issues.
    Keep `t.window.resizable = true` so the user can still toggle with Alt+Enter.
  </action>
  <verify>Launch `love norrust_love` and confirm it opens fullscreen</verify>
  <done>AC-1 satisfied: window fills entire screen on launch</done>
</task>

<task type="auto">
  <name>Task 2: Sort factions alphabetically after loading</name>
  <files>norrust_love/main.lua</files>
  <action>
    After line 555 (`factions = norrust.get_faction_ids(engine)`), add:
    `table.sort(factions, function(a, b) return a.name < b.name end)`

    This sorts the factions array in place by name. Since draw.lua iterates
    with ipairs(ctx.factions) and main.lua uses numeric indices for key input,
    the sorted order will be reflected everywhere consistently.
  </action>
  <verify>Launch game, pick any scenario, confirm faction list shows Elves, Loyalists, Orcs in that order</verify>
  <done>AC-2 satisfied: factions in alphabetical order on every launch</done>
</task>
