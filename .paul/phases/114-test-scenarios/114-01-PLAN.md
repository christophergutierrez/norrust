    In main.lua, after the debug_mode flag is set in the late arg-parsing block (where shared.debug_mode is set), prepend debug scenarios to game_data.SCENARIOS:

    ```lua
    if debug_mode then
        -- Prepend debug scenarios to the list
        table.insert(game_data.SCENARIOS, 1, {name = "[DBG] Advance Test", board = "debug_advance/board.toml", units = "debug_advance/units.toml", preset_units = true})
        table.insert(game_data.SCENARIOS, 2, {name = "[DBG] Recruit Test", board = "debug_recruit/board.toml", units = "debug_recruit/units.toml", preset_units = true})
    end
    ```

    This keeps existing scenarios accessible but puts debug ones first. The [DBG] prefix makes them visually distinct. preset_units = true skips faction picker for faster iteration.
