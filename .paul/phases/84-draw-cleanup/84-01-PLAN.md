    **2. Extract unit fallback circle helper:**
    Add a local helper near the top of draw.lua (after the existing helpers):
    ```lua
    local function draw_unit_fallback(ctx, cx, cy, faction, alpha, def_id, hp)
        local color = faction == 0 and ctx.BLUE or ctx.RED
        love.graphics.setColor(color[1], color[2], color[3], alpha)
        love.graphics.circle("fill", cx, cy, ctx.hex.RADIUS * 0.45)
        local word = (def_id or ""):match("^([^_]+)") or def_id or ""
        local abbrev = (word:sub(1,1):upper() .. word:sub(2):lower()):sub(1, 7)
        love.graphics.setColor(C_WHITE[1], C_WHITE[2], C_WHITE[3], alpha)
        love.graphics.setFont(ctx.fonts[14])
        love.graphics.printf(abbrev, cx - 42, cy - 14, 84, "center")
        love.graphics.setColor(C_WHITE[1], C_WHITE[2], C_WHITE[3], alpha)
        love.graphics.setFont(ctx.fonts[18])
        love.graphics.print(tostring(hp), cx - 12, cy - 2)
    end
    ```

    **3. Replace draw_units fallback (lines ~88-106) with helper call:**
    Replace the `if not drawn then ... end` block AND the HP rendering below it with:
    ```lua
    if not drawn then
        draw_unit_fallback(ctx, cx, cy, faction, alpha, unit.def_id, hp)
    else
        -- HP (always drawn on top of sprites)
        love.graphics.setColor(C_WHITE[1], C_WHITE[2], C_WHITE[3], 1)
        love.graphics.setFont(fonts[18])
        love.graphics.print(tostring(hp), cx - 12, cy - 2)
    end
    ```
    Note: HP is rendered by the helper for fallback mode, but still needs explicit rendering when sprites are drawn.

    **4. Replace ghost fallback (lines ~813-829) with helper call:**
    Replace the `if not drawn then ... end` block AND the HP below it with:
    ```lua
    if not drawn then
        draw_unit_fallback(ctx, gx, gy, faction, ghost_alpha, unit.def_id, hp)
    else
        love.graphics.setColor(1, 1, 1, ghost_alpha)
        love.graphics.setFont(ctx.fonts[18])
        love.graphics.print(tostring(hp), gx - 12, gy - 2)
    end
    ```

    **Important:**
    - The helper includes HP rendering so both fallback sites are fully replaced
    - Sprite path still needs its own HP rendering (sprites don't draw HP)
    - Use `ctx.fonts` not bare `fonts` (draw.lua accesses fonts through ctx)
